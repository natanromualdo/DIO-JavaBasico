# DIO - Java Basico
Repositório de Exercícios em Java Básico - DIO
Este repositório reúne exercícios desenvolvidos durante os estudos de Java Básico pela plataforma Digital Innovation One (DIO).

O objetivo é praticar e consolidar os principais conceitos da linguagem, incluindo:

- Sintaxe básica.
- Estruturas de decisão.
- Estruturas de repetição.
- Arrays.
- Métodos.
- Orientação a Objetos (introdução).

Os códigos foram implementados como forma de aprendizado contínuo, servindo também como material de consulta e evolução no desenvolvimento em Java.

Em constante atualização conforme o progresso nos estudos.
